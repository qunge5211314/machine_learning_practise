# Series(一维数据对象)
## Series简介
#### 定义
Series是一种类似于一维数组的对象，由一组数据和一组与之相关的数据标签(索引)组成。
#### 创建方式
```python
import pandas as pd
# 1.
pd.Series([4, 7, -5, 3])
# 2.
pd.Series([4, 7, -5, 3], index=['a', 'b', 'c', 'd'])
# 3.
pd.Series({'a': 1, 'b': 2})
# 4.
pd.Series(0, index=['a', 'b', 'c', 'd'])
```
* 获取值数组和索引数组：values属性和index属性。

* Series比较像列表(数组)和字典的结合体。
